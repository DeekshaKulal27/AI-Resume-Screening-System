# 🤖 AI Resume Screening System with Tracing

## 📌 Project Overview

This project is an AI-powered Resume Screening System that evaluates
resumes based on a given job description.

The system performs: - Skill Extraction - Resume Matching - Scoring
(0--100) - Explanation Generation

------------------------------------------------------------------------

## 🚀 Features

-   Extracts skills, tools, and experience from resumes\
-   Matches candidate profile with job requirements\
-   Assigns a score (0--100)\
-   Provides explainable output\
-   Includes debugging example\
-   Simulated tracing for pipeline visualization

------------------------------------------------------------------------

## 🧠 Pipeline Flow

Resume → Extract → Match → Score → Explain

------------------------------------------------------------------------
